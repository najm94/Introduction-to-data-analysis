Dealing with missing values

Since there are lot of missing values in Cabin column,we can't use it for the analysis process As for age values,since most of the data values lie within one standard deviations from the mean,we can fill those null values with random numbers between(mean-std) & (mean+std)

In [37]:
pd.options.mode.chained_assignment = None  
age_mean=tnc_df['Age'].mean()
age_std=tnc_df['Age'].std()
nan_age=tnc_df['Age'].isnull().sum()
rand=np.random.randint(age_mean-age_std,age_mean+age_std,size=nan_age)
tnc_df['Age'][np.isnan(tnc_df['Age'])] = rand

#Function to define child category
def male_female_child(passenger):
    
    age, sex = passenger
    
    if age <18:
        return 'child'
    else:
        return sex
tnc_df['Person'] = tnc_df[['Age', 'Sex']].apply(male_female_child, axis=1)

Final results and reflections

Conclusion

1.The survival rate reduces with age

2.Women and children are more likely to survive than men regardless of the passenger class or the port embarked

3.The passengers who were accompanied by a family were more likely to suvive than those who came alone

Limitations

Potential shortcomings of the data analysis process

1.Limitations of assumptions made without using statistical tests:The conclusions that women and children are more likely to survive or surviival rate reduces with age cannot be ascertained without using a statistical test so that we are assured that the results are not due to chance.Also,corelation does not imply causation i.e,the assumption that being a male passenger causes death is not true, but there is a definite relation or corelation between the two variables

2.Missing values in the dataset:There are lot of missing values in the Cabin column.So,we cannot be sure if it was factor which influenced the survival rate.Also,I filled the missing values in age column under the assumptions of the property of a normal distribution.

3.Limitation of linear regression used:Regression works well only if we have lot of data points.Also,these models are sensitive to outliers.More accurate predictions can be made by using machine learning techniques
